# Proyecto: Suma de Dos Números

Este proyecto es un programa sencillo en C++ que calcula la suma de dos números enteros utilizando una función.

## Descripción
El programa incluye una función llamada `suma` que toma dos números enteros como parámetros y devuelve la suma de estos. Los valores de los números a sumar están definidos dentro del programa, y el resultado se muestra en la consola.

### Características
- Uso de una función para realizar operaciones matemáticas.
- Salida del resultado directamente en la consola.

## Requisitos
Para compilar y ejecutar este programa, necesitarás:

- Un compilador de C++ (como `g++` o cualquier IDE que soporte C++).

## Ejecución
Sigue estos pasos para ejecutar el programa:

1. **Clonar o copiar el archivo fuente:** Guarda el código en un archivo llamado `suma.cpp`.

2. **Compilar el programa:** Ejecuta el siguiente comando en la terminal:

   ```bash
   g++ suma.cpp -o suma
   ```

3. **Ejecutar el programa:**

   ```bash
   ./suma
   ```

## Ejemplo de Ejecución
```
La suma de 5 y 7 es: 12
```

## Notas
- Los valores de los números (`num1` y `num2`) están definidos en el código como 5 y 7 respectivamente. Puedes modificar estos valores directamente en el programa para realizar sumas diferentes.

---

Este proyecto es ideal para practicar conceptos básicos de programación en C++, como el uso de funciones y manipulación de datos.
